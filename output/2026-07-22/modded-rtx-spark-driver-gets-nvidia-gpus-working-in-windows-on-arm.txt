Modded RTX Spark Driver Gets Nvidia GPUs Working in Windows on ARM

Modders have managed to get Nvidia's RTX Spark driver working on alternative Arm platforms, enabling Nvidia desktop GPUs to run on Windows on Arm systems, Windows Latest reports. Performance in game tests wasn't great, though the CPU is likely a major factor.
